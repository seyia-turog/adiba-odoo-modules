How to use the module: Project Scrum

Overview

This module is to provide the additional functionality over the default one as it did not provide from the project application
The project manager could easily manage the throught the scrum process as there are many additional field that you can put in each project such as Sprint Release, Sprint, User Story, Task, Project Documentation, Scrum Team, and the task

Features
- Manage the sprint release
- Manage the sprint in each release
- Manage the user story
- Manage the task in each sprint
- Manage the scrum meeting
- Manage the scrum Team
- Manage the project document
You can always feedback if you need more field or additional value to this module so that the team could add it for you.
We are always welcome and make it work.
